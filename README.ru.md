# Mqtt - Интеграция протокола MQTT

![Mqtt Icon](static/Mqtt.png)

Интеграция протокола MQTT для подключения к MQTT брокерам, подписки на топики и управления коммуникацией устройств.

## Описание

Модуль `Mqtt` предоставляет интеграцию протокола MQTT для платформы osysHome. Он позволяет подключаться к MQTT брокерам, подписываться на топики, осуществлять двустороннюю коммуникацию с устройствами и автоматически управлять топиками.

## Основные возможности

- ✅ **Подключение к MQTT брокеру**: Подключение к MQTT брокерам
- ✅ **Управление топиками**: Создание и управление MQTT топиками
- ✅ **Двусторонняя коммуникация**: Публикация и подписка на топики
- ✅ **Связывание свойств**: Связывание топиков со свойствами объектов
- ✅ **Связывание методов**: Связывание топиков с методами объектов
- ✅ **Автообнаружение**: Автоматическое обнаружение топиков
- ✅ **Замена значений**: Сопоставление значений между MQTT и объектами
- ✅ **Интеграция поиска**: Поиск топиков и связанных объектов

## Панель администратора

Модуль предоставляет комплексный административный интерфейс:

### Главный вид
- **Список топиков**: Просмотр всех настроенных топиков
- **Статус подключения**: Статус подключения в реальном времени
- **Настройки**: Настройка подключения к брокеру

### Конфигурация топика
- **Путь топика**: Путь MQTT топика
- **Заголовок**: Отображаемое имя
- **Связанный объект**: Связь с объектом osysHome
- **Связанное свойство**: Связь со свойством объекта
- **Связанный метод**: Связь с методом объекта
- **Замена значений**: Сопоставление значений MQTT с значениями объектов
- **QoS**: Уровень качества обслуживания
- **Retain**: Флаг сохранения сообщения
- **Только чтение**: Флаг топика только для чтения

## Управление подключением

### Настройки брокера
- **Хост**: Имя хоста/IP MQTT брокера
- **Порт**: Порт MQTT брокера (по умолчанию: 1883)
- **Логин**: Имя пользователя для аутентификации
- **Пароль**: Пароль для аутентификации
- **Автопереподключение**: Автоматическое переподключение при отключении

### Статус подключения
- Отображение статуса подключения в реальном времени
- Отслеживание попыток переподключения
- Отображение сообщений об ошибках
- WebSocket обновления статуса

## Использование

### Настройка брокера

1. Перейдите в модуль Mqtt
2. Введите хост и порт брокера
3. Установите аутентификацию при необходимости
4. Настройте топики подписки
5. Сохраните настройки

### Создание топика

1. Перейдите в модуль Mqtt
2. Нажмите "Add Topic"
3. Введите путь топика
4. Свяжите со свойством или методом объекта
5. Настройте замену значений при необходимости
6. Сохраните топик
